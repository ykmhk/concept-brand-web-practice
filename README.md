# NOTED.

A concept website for a fictional stationery brand focused on documenting everyday moments.

This project is a personal website building practice created to improve my skills in HTML, CSS, and JavaScript while exploring visual design, layout, animation, and responsive web development. 

## Disclaimer

This is a fictional brand created for learning and portfolio purposes. It is not associated with any existing company or commercial product.

## 🚧 This project is still a work in progress.

I plan to continue improving the website by adding more pictures, pages, refining animations, enhancing responsiveness, and implementing additional interactive features.

[](https://ykmhk.github.io/concept-brand-web-practice/)

## Assets & Credits

All pictures used in this project are downloaded from royalty-free image websites, including Pexels and Kaboompics. They are used for educational and portfolio purposes only and are not intended for commercial use.

The product names and descriptions are AI-generated placeholder content. They are included solely to support the visual presentation of the website, allowing this project to focus on front-end development and interface design.
